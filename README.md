# Zama Builder Track - Private Data App

## Mô tả
Dự án demo sử dụng FHEVM để xử lý dữ liệu nhạy cảm.

## Cách chạy
1. Cài đặt Node.js và Truffle.
2. Triển khai hợp đồng thông minh.
3. Chạy giao diện người dùng.
